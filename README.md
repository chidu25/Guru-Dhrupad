# Guru-Dhrupad

A Dhrupad master application powered by OpenRouter API.

## Description

Guru-Dhrupad is a Next.js application designed to explore and master the ancient Indian classical music form of Dhrupad. This app leverages the OpenRouter API to provide AI-powered assistance for learning and understanding Dhrupad music.

## Setup

1. Clone the repository:
```bash
git clone https://github.com/chidu25/Guru-Dhrupad.git
cd Guru-Dhrupad
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
   - Copy `.env.example` to `.env.local`
   - Add your OpenRouter API key to the `.env.local` file

4. Run the development server:
```bash
npm run dev
```

## Environment Variables

This application requires an OpenRouter API key. See `.env.example` for configuration details.

## Technologies

- Next.js
